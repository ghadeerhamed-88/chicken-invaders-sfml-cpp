# Chicken Invaders-style Game (C++ / SFML)

## Overview
This is a 2D arcade-style game developed using C++ and SFML, inspired by the classic Chicken Invaders game. The project demonstrates basic game development concepts such as real-time rendering, event handling, and collision detection.

## Features
- Built using C++ and SFML 
- Real-time game loop
- Player movement and shooting mechanics
- Enemy spawning system
- Collision detection (bullets vs enemies)
- Score tracking system
- 2 playable levels with increasing difficulty

## Controls
- Arrow keys: Move player
- Space: Shoot

## Technologies Used
- C++
- SFML
